# Causal Relationships among Ecology, Subsistence Strategies, and Social Organization in Prehistoric Farming Societies: A Bayesian Network Approach

**Rosy Oh** (Department of Mathematics, Korea Military Academy) and **Man-Suk Oh** (Department of Statistics, Ewha Womans University)

---

## Overview

This repository contains all R code used in the paper:

> Rosy Oh and Man-Suk Oh. "Causal relationships among ecology, subsistence strategies, and social organization in prehistoric farming societies: A Bayesian network approach." (2026)

The study investigates causal relationships among ecological factors, subsistence strategies, and social organization in 154 trans-historical, cross-cultural small-scale farming societies drawn from the D-PLACE and eHRAF databases. We apply a causal Bayesian network (BN) with mixed continuous and categorical variables, using the PC algorithm with hierarchical blacklist constraints, and quantify causal effects through Pearl's do-calculus via Monte Carlo simulation.

## Data

The analysis uses the cross-cultural ethnographic dataset of [Palacios et al. (2022)](https://doi.org/10.1371/journal.pone.0276088), publicly available via PLOS ONE. The main script downloads the dataset directly from the journal's supplementary materials; no manual download is required.

**Variables used:**

| Domain | Variables |
|---|---|
| Environmental (continuous) | DistanceCoast, Elevation, Slope, AnnualTemp, MonthlyPrecip, MonthlyPrimaryprod |
| Subsistence strategies (continuous) | Agriculture, AnimalHusbandry, Fishing, Hunting, Gathering |
| Social organization | CommSize (continuous), SettlementType, CommOrg, HouseholdOrg (categorical) |

After removing 18 cases with missing values and 1 case with an anomalous value (Lepcha), the final analytical sample comprises **154 societies**.

## Methods Summary

1. **Data preprocessing** — Interval-coded subsistence and community-size variables are converted to continuous scores via midpoint substitution. Coefficient-of-variation ecological variables are excluded to avoid multicollinearity. All continuous variables are Yeo–Johnson transformed and standardized (mean = 0, SD = 1).

2. **Causal structure learning** — The `tpc` algorithm (from the R package `pcalg`) is applied to mixed continuous–categorical data using kernel-based independence tests and discrete likelihood-ratio tests (α = 0.05), with hierarchical blacklist constraints encoding established socio-ecological theory.

3. **Bootstrap model averaging** — 2,000 bootstrap resamples (80% of cases, with replacement) are used to compute arc strength and direction probabilities. A consensus network is constructed by retaining arcs with strength ≥ 0.2.

4. **Interventional analysis** — Causal effects are estimated via stochastic forward simulation (1,000,000 Monte Carlo draws) from the fitted BN, implementing Pearl's do-calculus by severing incoming arcs to the intervened-upon variable. Results are summarized as interventional probability ratios comparing high vs. low levels of each ancestor variable.

## File Descriptions

### `Archeo_main.Rmd`

Main analysis script. Runs the full pipeline end-to-end:
- Package installation and loading
- Data download and preprocessing
- Normalization and standardization
- Blacklist construction
- BN structure learning with bootstrap aggregation
- BN visualization (node coloring, arc thickness by direction confidence)
- Local distribution fitting (Gaussian regression for continuous nodes; multinomial logistic regression for discrete nodes)
- Interventional analysis and plot generation

### `fn_normalize.R`

**Function:** `normalize(dat, cols)`

Applies Yeo–Johnson transformation (via `bestNormalize::yeojohnson`) to selected columns of a data frame. Returns the transformed data, a summary table of λ values and Shapiro–Wilk p-values before/after transformation, and the fitted transformation models.

### `fn_calculate_arc_strength.R`

**Function:** `calculate_arc_strength(boot_nets, nodes)`

Computes arc strength (proportion of bootstrap networks containing each edge, regardless of direction) and arc direction probability (proportion of occurrences pointing in a specified direction) from a list of bootstrap networks. Conflicting orientations within a single replicate are split equally.

### `fn_simulate_all_Y_interventions.R`

**Functions:**
- `simulate_all_Y_interventions()` — Outer wrapper: loops over all ancestor variables Y and all intervention levels, calls `simulate_doY_region()` for each combination, and collects results.
- `simulate_doY_region()` — Implements the do-operator: fixes Y to a specified level, then samples all remaining nodes sequentially from their fitted conditional distributions in topological order.
- `draw_Y_empirical()` — Samples values of Y from the observed data under a specified intervention level (used for continuous ancestors binarized at the median).

### `fn_interventional_analysis_ci.R`

**Functions:**
- `compute_interventional_analysis_ci()` — Main wrapper: computes interventional probabilities P(X | do(Y)) and 95% bootstrap confidence intervals for all X–Y ancestor pairs.
- `compute_interventional_prob_with_ci()` — Handles continuous Y (binarized at median).
- `compute_interventional_prob_with_ci_categorical_Y()` — Handles categorical Y (all factor levels).
- `create_results_table()` — Organizes results into two data frames (continuous Y / categorical Y).
- `plot_ratio()` — Produces grouped bar charts of interventional probability ratios, color-coded by direction of effect (increase/decrease).

## How to Reproduce the Analysis

### 1. Install R (≥ 4.2.0)

Download from [https://cran.r-project.org](https://cran.r-project.org).

### 2. Clone this repository

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
```

### 3. Create output directories

```r
dir.create("data",    showWarnings = FALSE)
dir.create("figures", showWarnings = FALSE)
```

### 4. Install required packages

The main script installs all required packages automatically on first run. Alternatively, install them manually:

```r
# Bioconductor packages
if (!require("BiocManager", quietly = TRUE)) install.packages("BiocManager")
BiocManager::install(c("graph", "RBGL", "Rgraphviz"))

# CRAN packages
install.packages(c(
  "readxl", "pcalg", "igraph", "bnlearn", "micd", "tpc",
  "nnet", "dplyr", "tidyverse", "GGally", "httr", "bestNormalize"
))
```

### 5. Run the main script

Open `Archeo_main.Rmd` in RStudio and click **Knit**, or run from the R console:

```r
rmarkdown::render("Archeo_main.Rmd")
```

The script will:
1. Download the raw dataset from PLOS ONE
2. Preprocess, transform, and standardize variables
3. Learn the causal BN structure with bootstrap aggregation
4. Save the BN graph to `figures/BN_thr2.png` and `figures/BN_thr2.pdf`
5. Fit local conditional distributions
6. Run interventional simulations (Monte Carlo, n = 1,000,000 per condition)
7. Save results to `data/interventional_results_*.csv`
8. Generate and display interventional probability ratio plots

> **Note:** Bootstrap aggregation (2,000 resamples) and Monte Carlo simulation (n = 1,000,000) are computationally intensive. Total runtime is approximately **90–120 minutes** depending on hardware. The intermediate simulation object `sim_results.rds` is saved automatically; uncomment the `readRDS` line to reload it without re-running the simulation.

## Key Results

The learned causal network (arc strength threshold = 0.2) reveals:

- **Ecological cascade:** Elevation → {Slope, AnnualTemp} → {MonthlyPrecip} → MonthlyPrimaryprod, consistent with geophysical principles.
- **Subsistence domain:** Agriculture and AnimalHusbandry (dominant strategies) act as upstream causes of Hunting, Fishing, and Gathering, with no environmental parent nodes — suggesting their primacy is sample-compositional rather than ecologically determined.
- **Novel direct pathways:** DistanceCoast → Hunting and Elevation → Fishing are detected as direct causal links bypassing intermediate climate/productivity layers.
- **Social organization:** SettlementType is shaped by nearly all ecological and subsistence variables. HouseholdOrg is influenced directly by terrain (Slope). CommOrg has DistanceCoast as its sole ancestor. CommSize is jointly determined by Agriculture and Gathering.
